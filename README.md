# Car-Parking-System 🐛

## To run this stuff, just follow this steps below on your terminal 👽️

First, generate vvp file:

```bash
iverilog -o sim.vvp tb_car_parking_system.v
```

Then, generate dump vcd file:

```bash
vvp sim.vvp
```

Lastly, simulate

```bash
gtkwave
```
